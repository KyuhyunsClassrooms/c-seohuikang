# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
시작
반복 (사용자가 '0'을 입력할 때까지):
    메뉴 출력 (1. 스트레칭 추천, 2. 스트레칭 목록 보기, 0. 종료)
    사용자 선택 입력
    만약 선택이 1이면:
        스트레칭 추천 함수 호출
    만약 선택이 2이면:
        스트레칭 목록 보기 함수 호출
    만약 선택이 0이면:
        반복 종료
반복 끝
종료


## 2. 핵심 기능 1: 

시작
    만약 사용자가 입력한 스트레칭 부위가 유효하지 않으면:
        "잘못된 입력입니다. 다시 입력하세요." 메시지 출력
        함수 종료
    아니면:
        사용자가 선택한 스트레칭 부위 저장
        스트레칭 부위에 맞는 스트레칭 목록을 출력
        사용자가 원하는 스트레칭을 선택하도록 유도
        만약 사용자가 선택한 스트레칭이 유효하면:
            선택한 스트레칭 이름과 설명 출력
            선택한 스트레칭의 권장 시간 출력
        아니면:
            "잘못된 선택입니다. 다시 선택하세요." 메시지 출력
        함수 종료
종료


## 3. 핵심 기능 2: 
```시작
    스트레칭 데이터를 저장할 배열 선언

    목 스트레칭 배열에 다음 문장 저장:
        1. "고개를 천천히 좌우로 돌리세요. (2분)"
        2. "고개를 앞뒤로 천천히 숙였다 드세요. (3분)"

    어깨 스트레칭 배열에 다음 문장 저장:
        1. "어깨를 천천히 앞으로 10회, 뒤로 10회 돌리세요. (3분)"
        2. "양팔을 뒤로 젖혀 가슴을 펴세요. (2분)"

    허리 스트레칭 배열에 다음 문장 저장:

        2. "양손을 허리에 두고 뒤로 젖혀 허리를 펴세요. (2분)"

    다리 스트레칭 배열에 다음 문장 저장:
        1. "무릎을 굽히고 발끝을 잡으며 10초 유지하세요. (3세트)"
        2. "앉은 자세에서 다리를 앞으로 쭉 뻗고 상체를 숙이세요. (2분)"

    모든 배열이 정상적으로 초기화되었는지 확인
    만약 초기화 실패 시:
        "데이터 초기화 실패" 메시지 출력
        함수 종료
    아니면:
        "스트레칭 데이터 준비 완료" 메시지 출력
        함수 종료
